# Back End Application 

To run the back end application

1) cd movie-backend
2) npm run start

To Test the front end application

1) cd movie-backend
2) npx jest

# Front End Application 

To run the front end application

1) cd movie-frontend
2) npm start

To Test the front end application

1) cd movie-frontend
2) npm run test
